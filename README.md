# StopWatch_SkillCraft


A clean, chronograph-style stopwatch built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just the fundamentals.

## 🔗 Live Demo

[View Live Site] :- https://niteshkumar395.github.io/StopWatch_SkillCraft/

## ✨ Features

- **Start / Pause / Reset** — full control over the timer
- **Lap tracking** — records split time (time since last lap) and total elapsed time
- **Fastest / slowest lap highlighting** — quickly spot your best and worst segments
- **Keyboard shortcuts** — `Space` to start/pause, `L` to lap, `R` to reset
- **Responsive design** — works smoothly on desktop and mobile
- **Accessible** — visible keyboard focus states and reduced-motion support

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no libraries or frameworks)

## 📁 Project Structure

```
├── StopWatch.html    # Markup
├── StopWatchcss     # Styling
└── StopWatch.js     # Stopwatch logic
```
